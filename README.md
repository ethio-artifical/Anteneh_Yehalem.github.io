## Run Locally

### Linux

```bash
sudo apt update
sudo apt install -y ruby-dev ruby-bundler nodejs build-essential gcc make
bundle config set --local path 'vendor/bundle'
bundle install
bundle exec jekyll serve -l -H localhost
```

Then open `http://localhost:4000`.

### Docker

```bash
docker compose up
```
